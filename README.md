## Supuesto práctico

AmazingCo es una empresa que se especializa en comercio electrónico. Uno de los aspectos más interesantes
es la innovación en las formas de envío, recepción y devolución de pedidos para los clientes de esta empresa.
Una de las nuevas apuestas de AmazingCo en este
campo es el denominado Package Locker. Son taquillas autoservicio situadas en diversos puntos de la ciudad que
posibilitan la recogida y devolución de pedidos con total
autonomía.
Estas taquillas suelen estar en el interior/exterior de
comercios , centros comerciales, estaciones de servicio u
oficinas de correo.
Cuando el cliente está tramitando su pedido, puede seleccionar un Package Locker como punto de recogida y
consultar el horario de apertura del comercio o lugar donde se encuentre y su ubicación. Los horarios de apertura
dependerán del punto de recogida y de la época del año.
Un empleado de AmazingCo lo llevará al punto de recogida seleccionado por el cliente, y lo introducirá en una
taquilla, asociando el código del paquete a un número de taquilla. Al cliente le llega una notificación con los datos
para recoger el paquete: el código del paquete y un enlace a la información sobre la ubicación y horario de apertura
del punto de recogida.
Cada paquete puede estar un número de días como máximo en la taquilla a la espera de ser recogido. Pasado
este período de tiempo, el cliente no podrá retirarlo y será devuelto a la central de AmazingCo. Se le notificará al
cliente nuevas opciones adicionales de recogida.
Cuando un cliente va a recoger un paquete en un Package Locker, deberá llevar el código proporcionado por
AmazingCo. Se aproximará al Package Locker e introducirá el código. Si el Locker comprueba que tiene un paquete
con el código introducido, abrirá automáticamente la taquilla que lo contiene. Si el código no corresponde a ningún
paquete o si el código corresponde a un paquete que ha excedido el número de días de almacenaje, se mostrará un
mensaje y no se abrirá ninguna taquilla.
